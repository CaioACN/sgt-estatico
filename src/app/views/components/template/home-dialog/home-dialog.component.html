Antes de começar a criar tarefas é fundamental cadastrar um responsável e um tipo de tarefa, respectivamente.

Para criar um comentário é necessário criar uma tarefa primeiro, para que você possa atribuir a mesma ao comentário.

As informações de cada campo seguem um padrão, caso os campos não sejam preenchidos de acordo com os exemplos, não serão salvos no sistema.
